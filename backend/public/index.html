GitHub Onboarding Generator

Generate a personalized onboarding path for any GitHub repository.

How to Run the Full Application

  1. Create a GitHub personal access token at https://github.com/settings/tokens
  2. Add the token to the .env file in the GITHUB_TOKEN field
  3. Open a terminal in the backend directory
  4. Run ./run_simple_form.sh to start the server
  5. Open http://localhost:8000 in your browser
